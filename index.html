Git Comandos

Ls-Mostra a lista de arquivos do local exemplo desktop.

Pwd-Mostra a onde você está

Cd / - leva para o local

Cd ../-sai da página.

Mkdir - Cria uma pasta e pode botar o nome

rm –r : Exclui uma pasta

Clear-limpa tudo na tela

Git clone –botar o code do repositório e levara os arquivos

Git status- mostra as alterações

Git add-envia os códigos para a pasta

Git commit –m = ele manda a descrição

Git push= ele envia os códigos.
